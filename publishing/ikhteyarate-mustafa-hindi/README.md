# Ikhteyarate Mustafa Hindi Publishing Project

This directory is for the Hindi/Devanagari transliteration manuscript of `Ikhteyarate Mustafa`.

The source remains:

`content/books/ikhteyarate-mustafa.json`

## Workflow

- This project uses manual batch transliteration.
- Do not use an automated Roman Urdu to Devanagari script for the manuscript.
- Continue section by section, similar to the English `Shifa Shareef` publishing workflow.
- Keep Urdu-Islamic wording and meaning intact; only change the script to Devanagari.
- Record style decisions in `notes/transliteration-style-guide.md`.
- Track progress in `manuscript/MIGRATION_STATUS.md`.
